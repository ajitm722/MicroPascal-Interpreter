# Interpreter jazyka Pascal

## Specifikace

Podmnožina jazyka Pascal, pro kterou je interpreter napsán, je vymazena takto:
- proměnné typů integer, boolean, string,
- procedury, funkce,
- while cyklus, for cyklus,
- if ... then ... else,
- writeln,
- binární operátory +, -, *, /, <, >, <=, >=, <>, ==, and, or, unární - a not.
